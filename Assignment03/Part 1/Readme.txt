
 Program is related to lexical analyzer (Differenctiate Identifiers , Numbers and Errors)

 On the start of Program put any mixture of string containing all possible characters on keyboard

	1. Identifiers are those who fulfill the requirment of variable names
	2. Numbers are those that contains only digits and are not eligible for identifiers
	3. Error are all those characters present on keyboard except identifier and Numbers

 DFA for Identifiers and Numbers are coded in program

 Program traverse all the input string and intelligently identify the portions.
