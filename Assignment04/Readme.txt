
	Lexical Analyzer

	1. Take Input from User using getline (incl. Spaces)
	2. Push the String into Lexical_Analyzer Function
	3. Function has Transition table of DFA for traverse the string on it
	4. It Differentiate the Following Accepting Strings

		* Simple Numbers
		* Fractional Numbers
		* Exponential Numbers
		* Spaces (' ',Tabs..)
		* Identifiers (Keyword,Variable Names)
		* Errors

	5. If string is not Accepted then It is a Partial String
