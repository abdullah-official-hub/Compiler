
 Convert any NFA (without absolone) into DFA

 Put into NFA.txt file

	1. No. of NFA states
	2. No. of Characters in character set
	3. Transition table of NFA

 Code is eligible to convert NFA (without absolone) into DFA

 Converted DFA is putted in DFA.txt

	A sample NFA.txt file exists in same folder.

 Note : Multiple names show a single state in DFA and Multiple names show in NFA means transition to that states.
